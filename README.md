## HelpDesk Unicon

Sistema de gestión de tickets de soporte técnico diseñado para optimizar el flujo de trabajo y la atención al usuario dentro de la empresa Unicon, ubicada en La Victoria, Aragua.

🚀 Descripción del Proyecto
Esta plataforma permite la centralización de incidencias técnicas, permitiendo que los usuarios reporten problemas, el personal técnico gestione las asignaciones y los administradores supervisen el rendimiento del departamento de TI en tiempo real.

🛠️ Tecnologías Utilizadas
El proyecto está construido bajo una arquitectura basada en tecnologías web estándar:

Backend: PHP (con manejo de sesiones seguro).

Base de Datos: MySQL (XAMPP).

Frontend: HTML5, CSS3, Bootstrap 5 (para una interfaz responsiva, moderna y ágil).

Interfaz: FontAwesome para iconos y componentes visuales personalizados.

📋 Características Principales
Gestión de Tickets: Creación, asignación y seguimiento de tickets por prioridad (Baja, Media, Alta, Crítica).

Dashboards Diferenciados: Vistas personalizadas según el rol del usuario (Admin, Técnico, Usuario).

Control de Rendimiento: Visualización de estadísticas y métricas para técnicos y administradores.

Interfaz Intuitiva: Diseño limpio y enfocado en la usabilidad técnica.

⚙️ Instalación y Configuración
Requisitos: Tener instalado XAMPP (o un servidor local similar).

Clonación: Coloca la carpeta en el directorio htdocs de tu servidor local.

Base de Datos: * Importa el archivo helpdesk_unicon.sql desde el panel de control de phpMyAdmin.

Verifica que la conexión en config/conexion.php coincida con tus credenciales de base de datos.

Acceso: Abre tu navegador y dirígete a http://localhost/helpdesk_unicon/index.php
